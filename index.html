Ray Tracing

For my advanced computer graphics class, I created a ray tracer to render scenes from Wavefront OBJ files (with some custom extensions) as an individual project. The original ray tracer is CPU based and written in C++. It uses material files for shading with basic functionality (Gouraud shading, shadows, and reflections).

As a group project for the same class, I implemented the same ray tracing functionality on a GPU using OpenCL.
